# Module 4: NVIDIA Isaac Platform

## Week 9: Perception and Manipulation in Isaac Sim

### Learning Objectives:
- Implement computer vision techniques for perception using Isaac Sim's sensor models.
- Develop robot manipulation strategies for grasping and object interaction.
- Utilize Isaac SDK components for common robotics tasks.

### Content:
- Camera, LiDAR, and depth sensor simulation.
- Real-time sensor data processing.
- Object detection and pose estimation within simulation.
- Inverse kinematics and motion planning for robotic arms.
- Task-oriented manipulation: pick-and-place, assembly.
