# Module 4: NVIDIA Isaac Platform

## Week 8: Introduction to Isaac SDK and Isaac Sim

### Learning Objectives:
- Understand the components and architecture of the NVIDIA Isaac SDK.
- Set up and navigate the Isaac Sim environment for robotics simulation.
- Learn basic robot manipulation and environment interaction within Isaac Sim.

### Content:
- Overview of the NVIDIA Isaac platform: SDK, Sim, Jetson.
- Installation and configuration of Isaac Sim.
- Importing and manipulating assets (robots, environments).
- Basic control of simulated robots.
- Introduction to Omniverse Kit and USD (Universal Scene Description).
