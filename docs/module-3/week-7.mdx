# Module 3: Robot Simulation with Gazebo

## Week 7: Physics, Sensor Simulation, and Unity Introduction

### Learning Objectives:
- Simulate realistic physics and sensor data within Gazebo.
- Integrate ROS 2 with Gazebo for robot control and data exchange.
- Get an introduction to Unity as an alternative simulation platform for robotics.

### Content:
- Gazebo physics engine: dynamics, contact, friction.
- Simulating camera, lidar, IMU, and other sensors.
- ROS 2 Gazebo plugins for seamless integration.
- Basic robot control in simulation.
- Introduction to Unity: editor, assets, scripting for robotics applications.
