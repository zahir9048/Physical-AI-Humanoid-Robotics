# Module 3: Robot Simulation with Gazebo

## Week 6: Gazebo Setup and URDF/SDF Models

### Learning Objectives:
- Set up and configure the Gazebo simulation environment.
- Create and integrate URDF (Unified Robot Description Format) models for robot description.
- Understand SDF (Simulation Description Format) for world and model definition in Gazebo.

### Content:
- Installation and basic usage of Gazebo.
- Introduction to URDF: links, joints, transmissions, sensors.
- Building complex robot models with URDF.
- SDF for environmental elements and advanced simulation features.
- Loading and interacting with robot models in Gazebo.
